在ubuntu系统中，常用的安装软件命令有apt和pkg。当然，还有一种更为简单的方式。那就是一个叫做tasksel的东西。

tasksel是一个易于使用的软件安装器，他可以快速的安装例如LAMP服务器，邮件服务器 ，DNS服务器之类的软件，有点像宝塔一键安装软件套装一样。

安装tasksel

要安装tasksel，只需要一个指令：
apt-get install tasksel -y
安装Tasksel后，它可以安装一个或多个预定义的软件包组。 用户需要使用几个参数从命令行运行它，当然，它也提供了一个图形用户界面，可以选择要安装的软件。

表达的方法

问题：集合｛x | -5x+6=0｝还可以用哪一种方法来表示呢？它可以表示为｛2，3｝吗？或者｛1，2，3｝？
看过了上一篇文章的同学应该都知道，可以把集合｛x | x²-5x+6=0｝用列举法可以表示成为（2，3）由此我们可以回答上面的问题：-2∉｛2，3｝，2∈｛2，3｝而且因为集合元素的无序性，即使我们写成（3，2）也不会影响最后的结论。

我们再来看一下：｛x = x²-5x+6=0｝=｛2，3｝
我们可以看到等号左边是用描述法，等号右边是列举法。那我应该怎样选择描述的方法呢？

描述法：可以看到集合的本质。
列举法：可以非常直观的表达解集。

数学的集合概念

发表于 2021-10-31 更新于 2021-11-06 分类于 学习 Valine：

集合的概念

勇哥上数学课，一向都是生涩难懂的。￼我搞了好久才理解，才搞明白这些到底是什么意思。￼以下就是我的见解。

集合的表达。

如何简洁，准确地表达数学对象及研究范围：
举个栗子：
(1)1-11之间的偶数。￼
(2)xxx中学今年入学的全体高一新生。￼

那么在(1)中我们把1-11间的每一个偶数。2，4，6，8，10作为研究对象。
一样的，我们把xxx中学￼￼￼的每一个高中生作为研究对象。￼
知识点：
一般的，我们把研究对象统称为元素(element)，把一些元素组成的整体。￼叫做集合(set)，或者简称集。
对一个给定的集合，有：
集合具有确定性。￼
集合元素具有互异性。￼
结合元素具有共存性。￼
